# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2023-10-28

### Added
- Advanced LinkedIn post generation logic following professional standards.
- New input fields: Audience, Length, CTA preference.
- Expanded Post Types: Thought Leadership, Personal Story, Career Update, Lesson Learned, Announcement, Hiring, Educational, and Opinion.
- Multi-part post structure (Hook, Body, CTA).
- Support for alternate hooks and post variations.
- Option to include/exclude hashtags.
- Modern LinkedIn-inspired UI with improved responsiveness.
- Clarification prompts for vague or very short inputs.

### Changed
- Refactored `script.js` for modularity and more complex generation logic.
- Updated `style.css` with a cleaner, professional look.
- Updated `README.md` to reflect the new feature set.

## [1.1.0] - 2023-10-27

### Added
- "Copy to Clipboard" feature for generated posts.
- SEO meta tags (description and keywords) in `index.html`.
- `AGENTS.md` to provide context and instructions for AI agents.

### Changed
- Improved layout responsiveness in `style.css`.
- Updated GitHub Actions dependencies in `static.yml`.
- Updated `README.md` with new features and instructions.

## [1.0.0] - 2023-10-26

### Added
- Initial release of the LinkedIn Post Generator.
- Features: Post idea input, tone selection, format selection.
- Deployment via GitHub Pages.
