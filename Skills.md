# Technical skills: PostGenius

This project demonstrates proficiency in modern, vanilla web development. Below are the core competencies used to architect PostGenius.

## Frontend architecture
- **Vanilla ES6+ mastery**: Advanced DOM manipulation, event delegation, and state management without frameworks.
- **ES Modules (ESM)**: Implementing a modular, future-proof codebase that works natively in the browser and across testing environments.
- **Real-time DOM synchronization**: Implementing live-update systems for instant LinkedIn mockup previews.
- **Dynamic asset management**: Orchestrating client-side file reading for custom avatar uploads.
- **Google coding standards**: Strict adherence to the **Google JavaScript Style Guide**, **Google HTML/CSS Style Guide**, and **Google Developer Documentation Style Guide**.
- **Asynchronous logic**: Managing clipboard APIs and simulated generation delays for a premium experience.

## Modern styling and UX
- **CSS3 variables and HSL**: Using a design system built on HSL color functions for consistent theming and easy dark mode implementation.
- **Glassmorphism and filters**: Implementing `backdrop-filter` and advanced shadow systems to achieve a high-fidelity UI.
- **Skeleton generation**: Architecting multi-layered shimmer animations to manage user expectations during simulated processing.
- **Confetti particle systems**: Implementing physics-based animation engines for high-impact success feedback.
- **Responsive design**: Mastering CSS Grid and Flexbox to create a two-pane layout that scales from mobile to wide-screen monitors.
- **Micro-animations**: Using keyframes and cubic-bezier transitions to create an interactive interface.

## Data and persistence
- **Client-side persistence**: Efficiently using `localStorage` for semi-permanent data storage with JSON serialization.
- **Robustness and error handling**: Implementing `try-catch` wrappers and input sanitization to prevent data corruption and security vulnerabilities.

## Testing and quality assurance
- **Unit testing with Vitest**: Architecting logic to be purely functional and testable, ensuring high reliability of the generation engine.
- **Enterprise-grade CI/CD**: Implementing a robust GitHub Actions pipeline with automated **Quality Gates** (Linting and Unit Testing) before deployment.
- **Automated linting**: Utilizing ESLint with custom configurations to enforce code quality across the entire project.

## Content strategy
- **Framework implementation**: Translating proven LinkedIn marketing strategies into programmatic templates.
- **Tone algorithms**: Creating dynamic string replacement logic to transform professional text into casual variants in real-time.
